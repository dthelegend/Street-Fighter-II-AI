# Street-Fighter-II-AI

## Install

```bash
conda env create -f environment.yml
```

```bash
poetry install
```

```bash
python3 -m retro.import roms/
```


## Notes

Controls are mapped to the buttons using a binary format

| 128 | 64 | 32 | 16 | - | 8 | 4 | 2 | 1 |
| --- | -- | -- | -- | - | - | - | - | - |
|  -  |  C |  B |  A | - | - | Z | Y | X |
